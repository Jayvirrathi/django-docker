## Docker Django

### Docker Image

```
docker-compose run web django-admin startproject myfirstapp .

docker-compose up
```

### Docker Create new App

Create New App (Keep running docker-compose up command)

```
docker exec django-docker_web_1 python manage.py startapp demoapp
```
